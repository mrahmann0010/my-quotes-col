Quick Start

Fetch your first quote in seconds — no API key required.

const res = await fetch(
  'https://my-quotes-col.onrender.com/quotes/random'
);
const quote = await res.json();
console.log(quote.text); // বাংলা উদ্ধৃতি
curl -X GET \
  https://my-quotes-col.onrender.com/quotes/random \
  -H "Accept: application/json"
import axios from 'axios';

const { data } = await axios.get(
  'https://my-quotes-col.onrender.com/quotes/random'
);
console.log(data.author); // Rabindranath Tagore

Notes & Conventions

◈

All responses are JSON. Bengali text is returned in UTF-8 Unicode — no additional encoding is required.

◈

The API is hosted on Render's free tier. Cold starts may cause the first request to take 10–30 seconds after a period of inactivity. Subsequent requests are fast.

◈

There is currently no authentication required for read endpoints. Write endpoints (POST /quotes) are open for community submissions.

◈

The frontend at BanglaQuotes proxies all requests through /api/* rewrites to this base URL, so both environments are always in sync.
